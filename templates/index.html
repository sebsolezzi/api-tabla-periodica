Tabla periodica - API

Mi primera API funcional que usa solo metodo el HTTP GET. La idea de mi proyecto surgió al conocer que Bill Gates tiene montada en la pared de su oficina una tabla periodica gigante con muestras o representaciones de todos los elementos quimicos.

/todos

Regresa un JSON con todos los elementos de la tabla periódica.

/simbolo/<simbolodelelemento>

Regresa un solo elemento de la tabla periódica. Por ejemplo, si deseara buscar el elemento cobre su consulta debería ser /simbolo/Cu

/electrones/<numelectrones>

Regresa un solo elemento de la tabla periódica. Si desea buscar el elemento posea que 79 como número atómico su consulta debería ser /electrones/79

/serie/<nombredeserie>

Regresa un JSON con los elementos de la tabla periódica que concuenden con esa serie química. Si quisiera consultar sobre los elemento de tipo no metales su consulta debería ser /serie/nometales
